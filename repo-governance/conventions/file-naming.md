# File & Identifier Naming

- **Feature/epic/lib names:** kebab-case (`platform-foundation`, `threat-intel`, `@vacti/db`).
- **Task files (ccpm):** sequential `001.md`, `002.md` before sync; renamed to issue number after.
- **Markdown docs:** kebab-case (`getting-started.md`).
- **TS files:** kebab-case for modules; PascalCase for React components; `*.test.ts`/`*.spec.ts`
  for unit, `*.integration.test.ts`, `*.e2e.ts` for the other tiers.
- **DB:** snake_case tables & columns.
- **Env vars:** SCREAMING_SNAKE_CASE.
