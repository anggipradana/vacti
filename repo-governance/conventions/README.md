# Layer 2 - Conventions

Documentation and style rules. See individual files:

- [diataxis-framework.md](diataxis-framework.md) - how docs are organized
- [file-naming.md](file-naming.md) - naming rules
- [commit-conventions.md](commit-conventions.md) - Conventional Commits + scopes

## At a glance

- **Markdown:** dash bullets, 2-space nested indent, blank lines around headings/code. Linted by
  markdownlint-cli2.
- **Code style:** Prettier `printWidth: 120`, single quotes, semicolons, trailing commas. ESLint for
  correctness.
- **Language:** planning docs may be in Indonesian; code identifiers and code comments in English.
