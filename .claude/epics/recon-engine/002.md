# Task: Tool runner framework

## Description
Kerangka generik menjalankan binary recon dari worker: spawn child-process, stream stdout (JSONL),
timeout per-tool, audit ke `commands`, dan **cancellation** via `AbortController` + kill process-group.

Menutup baris Checklist: 1.16, 1.18.

## Acceptance Criteria
- [ ] `runTool({bin, args, timeoutMs, signal, onLine})` → spawn, stream baris, kumpulkan exit code.
- [ ] Cancellation: `AbortController` membunuh seluruh process-group < 5 dtk; status tercatat.
- [ ] Timeout per-tool → kill + tandai timeout (bukan crash pipeline).
- [ ] Setiap eksekusi menulis `commands` (argv, exit_code, durasi) + simpan output mentah ke file.
- [ ] Deteksi binary hilang/at versi salah → error jelas (fail-fast, actionable).
- [ ] Unit test (mock proses), integration test (binary `echo`/dummy) untuk cancel & timeout.

## Technical Details
- Gunakan `child_process.spawn` dengan `detached`+`process group` untuk kill bersih.
- Backpressure aman untuk output besar; parsing diserahkan ke adapter (task 003/004).

## Dependencies
- platform-foundation #001 (workspace), #008 (binari di image, untuk integration nyata).
